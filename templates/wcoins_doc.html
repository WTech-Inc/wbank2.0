快速開始

安裝 web3.js 然後複製貼上就得

BASH 
npm install web3

1. 連接 RPC

初始化 Web3 並檢查連接

WEB3.JS 
import Web3 from "web3";

const web3 = new Web3(
  "https://wbank.wtechhk.com/wcoins/api/rpc"
);

const chainId = await web3.eth.getChainId();
console.log(`Chain: ${chainId}`); // 425

const block = await web3.eth.getBlockNumber();
console.log(`Block: ${block}`);

2. 查詢餘額

用 username 或 0x 地址查

WEB3.JS 
import Web3 from "web3";

const web3 = new Web3("https://wbank.wtechhk.com/wcoins/api/rpc");

// 用 username
const bal1 = await web3.eth.getBalance("ben");
console.log(`ben: ${Number(bal1)} WCN`);

// 用 0x 地址
const bal2 = await web3.eth.getBalance(
  "0x6700869c8ff7480e34a70a708b028700dbaa3a03"
);
console.log(`ben: ${Number(bal2)} WCN`);

3. 發送 WCoins

用 raw RPC call 發送交易（Gas = 0）

WEB3.JS 
import Web3 from "web3";

const web3 = new Web3("https://wbank.wtechhk.com/wcoins/api/rpc");

// Address mapping
const ADDR = {
  wangtry: "0x177876F0aCebc209105e4e4eAB9F8B19263C76E2",
  ben:     "0x6700869c8ff7480e34a70a708b028700dbaa3a03",
  wtech:   "0xd7ebbbb5e2bd3825fc3f41494067f26185e8ba5a",
};

async function send(from, to, amount) {
  const result = await web3.eth.requestManager.send({
    method: "eth_sendTransaction",
    params: [{
      from: ADDR[from] || from,
      to: ADDR[to] || to,
      value: "0x" + (BigInt(amount) * 10n ** 18n).toString(16),
      gas: "0x5208",
      gasPrice: "0x0",
      type: "0x0"
    }]
  });
  return result;
}

// 發送 10 WCN
send("wangtry", "ben", 10)
  .then(hash => console.log(`TX: ${hash}`))
  .catch(e => console.log(`Error: ${e.message}`));

4. Python Web3

用 web3.py 連接同查詢

WEB3.PY 
from web3 import Web3

w3 = Web3(Web3.HTTPProvider(
    "https://wbank.wtechhk.com/wcoins/api/rpc"
))

# 檢查連接
print(f"Chain: {w3.eth.chain_id}")  # 425

# 查詢餘額
bal = w3.eth.get_balance("ben")
print(f"ben: {bal} WCN")

# 查 Block
block = w3.eth.block_number
print(f"Block: {block}")

5. REST API

唔用 web3 嘅話可以直接 call REST API

Send
POST /wcoins/api/send application/json
Balance
GET /wcoins/api/balance?address=ben
History
GET /wcoins/api/history?address=ben
RPC
POST /wcoins/api/rpc JSON-RPC
CURL 
# 查餘額
curl "https://wbank.wtechhk.com/wcoins/api/balance?address=ben"

# 發送 WCoins（需登入）
curl -X POST "https://wbank.wtechhk.com/wcoins/api/send"   -H "Content-Type: application/json"   -d '{"receiver":"wangtry","amount":10}'

# RPC call
curl -X POST "https://wbank.wtechhk.com/wcoins/api/rpc"   -H "Content-Type: application/json"   -d '{"method":"eth_blockNumber","params":[],"id":1}'
